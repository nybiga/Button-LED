# Button-LED

This project uses PlatformIO as it's ecosystem. Depending on the state of the LED when the button LED ligit is turned on or off.

## Pins and corresponding segments
Button = 2
LED = 3

## Tools used

An Arduino compatible board
A Breadboard
10k ohms resistor
330 ohms resistor
Jumper wires(6)
